# map
```js
> let myArray = [1,2,3,4,5]
undefined
> myArray.map(v => v**2);
[ 1, 4, 9, 16, 25 ]
> myArray
[ 1, 2, 3, 4, 5 ]
```
# filter
```js
> let myArray = [1,2,3,4,5]
undefined
> myArray.filter(n => n%2 === 0)
[ 2, 4 ]
> myArray.filter(n => n%2 === 1)
[ 1, 3, 5 ]
> myArray
[ 1, 2, 3, 4, 5 ]
```
