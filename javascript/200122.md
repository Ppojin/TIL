# HTML

## 함수
- () = method in java
- () = function in javascript
## 특수문자
`\'` = '

`\\` = \

`\n` = enter

`\t` = tab

## +
- 덧셈
    ```js
    a = 1+2;
    ```
- 문자열 결합
    ```js
    a = "안녕";
    b = "하세요";
    x = a + b;
    ```
    x 의 결과 = `안녕하세요`
